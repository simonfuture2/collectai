# Grading Rubric

Maps a visual pre-grade assessment to a likely grade band. This is a *pre-grade* aid only — it estimates where a card might land, not a guarantee. Always present pre-grades as a range, never a single certain grade.

Read this file before assigning any pre-grade in step 3 of the workflow.

## Assessment dimensions

Evaluate four dimensions from the image. Each is scored on the band it most clearly falls into; the overall band is the *lowest* of the four (graders punish the worst flaw, not the average).

### Centering
- **Gem (10)** — 55/45 or better on both axes
- **Mint (9)** — up to 60/40
- **Near-mint (7–8)** — up to 65/35
- **Below (≤6)** — worse than 65/35 or visibly off

### Corners
- **Gem (10)** — all four sharp under magnification
- **Mint (9)** — sharp to the naked eye, minor softness under magnification
- **Near-mint (7–8)** — one or two corners with light wear
- **Below (≤6)** — visible rounding or dinging

### Edges
- **Gem (10)** — clean, no chipping or whitening
- **Mint (9)** — trace whitening on one edge
- **Near-mint (7–8)** — whitening on multiple edges
- **Below (≤6)** — chipping or notching

### Surface
- **Gem (10)** — no print lines, scratches, or indentations
- **Mint (9)** — one trivial flaw visible only at an angle
- **Near-mint (7–8)** — light scratching or a visible print line
- **Below (≤6)** — scratches, stains, or creases

## Reporting

Report the band, the limiting dimension, and the range. Example:

> Pre-grade: PSA 8–9 (limited by edges — whitening visible on the right border)

If image quality is too low to assess a dimension, say so and widen the range accordingly. Never report a confident single grade from a single photo.
